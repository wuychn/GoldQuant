## 策略说明

本系统**仅执行主升浪战法**，已移除涨停板战法、龙回头战法。

---

## 主升浪战法定位

- **标的**：主线题材中的**龙头股**（概念共振 + 人气靠前），不是单纯波动票。
- **趋势**：均线多头发散向上（MA5 > MA10 > MA20，间距扩大），走主升浪而非震荡。
- **买点**：
  1. **上升途中**：均线发散 + 价在 MA5 上方 + 分时均价线上方；
  2. **回调企稳**：趋势未破 MA20，回踩 MA10/MA20 区间后企稳收强。
- **卖点**（分两类，各自独立三确认链）：
  1. **破5日线**：当日有效跌破 MA5（较快退出）；
  2. **趋势衰竭**：MA5 下穿 MA10、多头结构破坏、跌破 MA20、MACD 转弱等（可跨日逐步确认）。
- **卖出时点（神奇2点30）**：
  - 第 1、2 次确认可在 **14:30 前**产生；
  - **第 3 次（最终确认）**须在 **14:30 之后**（通常 ~14:40）才完成并允许成交；
  - **止损**不受此限。
- **主线**：非当日单榜。近 5 日内至少 2 天上榜，且窗口内至少 1 次资金流入榜确认（见 `main_theme` 配置）。

加自选：晚间复盘，候选来自人气榜 + 主线概念；须通过主升浪评分维度。

买入前提：标的已在自选股中。

---

## 三确认机制

买卖均须 **连续 3 次**同向信号才执行成交：

| 次数 | 含义 |
|------|------|
| 第 1 次 | 记录信号 |
| 第 2 次 | 确认 |
| 第 3 次 | 再确认 → 允许交易 |

相邻两次信号间隔须在 **[min_interval, max_interval]** 分钟内（过近忽略，过久重置）。

间隔按**市场状态**（强势/震荡/弱势）分别配置；**破5日线 / 趋势衰竭** 可在 `confirmation.by_kind` 单独设间隔（趋势衰竭默认 max 72h）。可由 ML 离线校准。

## 共用约束

标的池：沪深主板 60/00、创业板 30；排除 ST、科创板、北交所。
T+1；止损冷却 3 日；极端熔断、每日亏损限额见 `quant/config/quant.yml` gates 段。

## 仓位联动

见 `quant/config/quant.yml` gates.position — 仅「主升浪战法」单票上限。

---

## 仓位与三确认档位

程序内部由 `大盘指数`、`赚钱效应`、`涨停统计` 推断强势/震荡/弱势，用于 `quant/config/quant.yml` 中的仓位上限与三确认间隔；**不在 API JSON 中单独返回**。
